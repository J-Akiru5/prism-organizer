What Prism Organizer is for

Prism Organizer is a Windows-focused CLI and TUI tool that scans, analyzes, and organizes files. It helps users clean up folders by sorting files into categories or date buckets, finding duplicates, and cleaning junk. It is designed with safety first: previews, backups before delete, and undo logs.

Core user outcomes

  • Understand a folder quickly (scan report).
  • Organize files by type or date safely.
  • Remove junk and duplicates with review.

Key safety features

  • Dry-run previews before changes.
  • Backups for deletes (undo support).
  • Cloud-drive detection (needs wiring).

Audience: end users who want safer file cleanup and organization on Windows.

Scope and priorities

Target: open source readiness with clean repo, safety behavior matching docs, and basic governance files.

Must ship

  • Cloud-drive detection wired into CLI and TUI.
  • Remove local artifacts and dev-only scripts.
  • Governance docs (contrib, code of conduct, security).

Should ship

  • Version consistency across Python and npm.
  • Checksum enforcement in the npm wrapper.
  • CI for tests on Windows.
